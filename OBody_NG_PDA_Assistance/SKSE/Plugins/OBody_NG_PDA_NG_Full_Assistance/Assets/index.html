Quick JSON Modifier

In this tab, you will be able to modify the master JSON on your own, safely; consider it a sandbox, since what you will modify is a copy of the updated file. And when you consider it ready, the "transfer configuration to game" button will perform the update. Here there is backup and more. It is recommended to use it with the game running, with Skyrim open, so you can browse the NPC list, get the ideal IDs with a few clicks, it's fun and took me time to make, so enjoy. It's a mix of C++, JS, and Python. Gradually, I'm getting the hang of this mixer system; it's nicer than just using Python with Qt5. And remember, all changes applied in this area will be reflected in Skyrim next update, in the next session when you reopen the game. This is OBody NG thing, so I can't do much about it.

Port Configuration Current Port: Loading...

Configure your port personally, only recommended in case you already use the default port

You can choose from 0 to 65535 as a port, but remember that the PC generally uses those from 0 to 1024 by default (used for printers, LAN ports, etc.). For me, ports 6050, 6051... are always free, but it's your decision. I only recommend changing this port if 6050 is already being used (which has a probability of 0.001526%) and you need to use another. Never use 8080, 49912, 8020, 7860...
